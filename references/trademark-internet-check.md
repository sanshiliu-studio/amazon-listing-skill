# Trademark and Brand Internet Check

Use this reference when the user provides keywords, competitor copy, product copy, brand names, model names, series names, compatibility terms, or any text that may contain third-party trademarks.

## When to Search

Search the internet before drafting or finalizing if any user-provided text includes:

- Capitalized product or brand-like terms.
- Phone, appliance, electronics, automotive, printer, tool, toy, or accessory model names.
- Competitor ASIN copy, marketplace titles, or competitor bullet points.
- Terms that look like platform, manufacturer, product-line, technology, certification, or proprietary feature names.
- Keywords the user asks to embed exactly.

Do not rely only on memory for trademark status. Trademark and brand ownership can change, and internet verification is part of this skill's workflow.

## How to Search

Use web search with combinations like:

```text
"TERM" brand
"TERM" trademark
"TERM" official
"TERM" Amazon
"TERM" manufacturer
```

Prefer official brand/manufacturer pages, Amazon product pages, USPTO/TMview or other trademark registry pages, and reputable retailer/category pages. If sources conflict, treat the term as risky and surface it in Review Notes.

## Decision Rules

- If the term appears to be another company's brand, trademark, product line, model, or proprietary technology, do not use it as if it describes the seller's own product.
- If the seller's product is genuinely compatible with that branded product, use clear compatibility wording such as `compatible with`, `for`, `fits`, `designed for`, or `intended for`.
- Put compatibility language in the title only when it is necessary for shopper understanding and accurate to the product.
- Do not include third-party brand names, product lines, model names, or ASINs in backend search terms.
- Do not use another brand name in a way that implies source, sponsorship, affiliation, endorsement, or official status.
- If compatibility is uncertain, ask the user to confirm before using the term in listing copy.

## Output Pattern

When a third-party brand term is needed, prefer:

```text
Brand Case compatible with iPhone 16 Pro
Replacement filter for Dyson V11 vacuum
Charging cable designed for Samsung Galaxy devices
```

Avoid:

```text
iPhone Case
Dyson V11 Filter
Samsung Charging Cable
```

## Review Notes

Briefly list any checked brand/trademark-like terms and the action taken:

```text
Trademark note: "iPhone" appears to be an Apple brand term, so the title uses "compatible with iPhone" wording and the term is excluded from backend search terms.
```

Make clear that internet checks are directional and do not replace seller legal review, Amazon Brand Registry review, or account-health review.
